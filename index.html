About the Game

rush.io is a fast-paced FPS inspired by classics like krunker.io. The focus is on pure skill, tight movement, and clean gunplay. The game features bhop mechanics for momentum-based movement and fluid traversal. Expect intensity and chaos in every match.

Key Features

  • Smooth bhop-based movement mechanics
  • Quick respawn gunfight gameplay
  • Simple and clean UI
  • Multiple maps in rotation
  • Real-time kill logs and stats
  • Optimized for performance
